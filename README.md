ci - Настройка CI/CD
fix - Исправление бага
feat - Новая функциональность
docs - Изменения в документации
config - Изменения в конфигурации
perf - Улучшение производительности
revert - Отмена предыдущего коммита
test - Добавление или изменение тестов
build - Изменения в системе сборки или внешних зависимостях
refactor - Рефакторинг кода (не исправляет баги и не добавляет фичи)
style - Форматирование, пробелы, точки с запятыми (не влияет на логику)
chore - Обновление зависимостей, настройка сборки (не влияет на код приложения)
