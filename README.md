Тип Описание Пример
feat Новая функциональность feat: add dark theme support
fix Исправление бага fix: correct typo in error message
docs Изменения в документации docs: update API documentation
style Форматирование, пробелы, точки с запятыми (не влияет на логику) style: fix indentation in components
refactor Рефакторинг кода (не исправляет баги и не добавляет фичи) refactor: extract validation logic to separate file
perf Улучшение производительности perf: optimize image loading
test Добавление или изменение тестов test: add tests for login flow
chore Обновление зависимостей, настройка сборки (не влияет на код приложения) chore: update vite to v5
build Изменения в системе сборки или внешних зависимостях build: configure webpack
ci Настройка CI/CD ci: add GitHub Actions workflow
revert Отмена предыдущего коммита revert: revert commit abc123
config Изменения в конфигурации (ваш случай!) config: update tailwind colors
